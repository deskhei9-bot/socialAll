üöÄ Quick Start Guides

Getting Started with SocialFlow

Access the Platform: Open your browser and navigate to your SocialFlow instance at https://socialautoupload.com
Register an Account: Click "Sign Up" and create your account with email and password
Connect Social Media: Go to Settings ‚Üí Social Accounts and connect your Facebook, Instagram, YouTube, TikTok, Twitter, or LinkedIn accounts
Create Publishing Profile: Set up your first publishing profile with default settings for your posts
Create Your First Post: Navigate to "New Post", upload media, write caption, and publish or schedule

Connecting Social Media Accounts

Navigate to Settings: Click the gear icon in the sidebar or go to /settings
Select Platform: Choose the social media platform you want to connect (Facebook, Instagram, YouTube, etc.)
Authorize Access: Click "Connect" and log in to your social media account when prompted
Grant Permissions: Allow SocialFlow to post on your behalf (permissions vary by platform)
Verify Connection: You should see "Connected" status with your account name displayed

üîß Troubleshooting

Common Issues & Solutions

Problem Possible Cause Solution
Cannot log in Incorrect credentials or expired session Reset password or clear browser cookies and try again
Social media won't connect OAuth permissions denied or expired tokens Disconnect and reconnect the account. Check platform app settings.
Upload fails File too large or unsupported format Max 100MB. Supported: JPG, PNG, MP4, MOV. Compress large files.
Post failed to publish Token expired or platform API issue Reconnect account. Check platform status. Try again later.
Scheduler not working Server timezone mismatch or cron job disabled Check server time. Ensure cron is running. Contact admin.
AI features not working OpenAI API key missing or quota exceeded Check OPENAI_API_KEY in .env file. Verify API quota.
500 Internal Server Error Backend service down or database connection issue Check backend logs: pm2 logs. Restart: pm2 restart socialflow-backend

Quick API Reference

Authentication

POST /api/auth/register - Register new user

POST /api/auth/login - Login and get JWT token

GET /api/auth/me - Get current user info

Posts

POST /api/posts - Create new post

GET /api/posts - Get all user posts

GET /api/posts/:id - Get specific post

DELETE /api/posts/:id - Delete post

Publishing Profiles

GET /api/publishing-profiles - Get all profiles

POST /api/publishing-profiles - Create profile

PUT /api/publishing-profiles/:id - Update profile

DELETE /api/publishing-profiles/:id - Delete profile
