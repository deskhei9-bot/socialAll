# Publishing Profiles - အသုံးပြုပုံလမ်းညွှန် (Myanmar)

## ခြုံငုံသုံးသပ်ချက်

Publishing Profiles သည် သင်၏ အကြိုက်ဆုံး channel များကို တစ်ခါတည်း သိမ်းဆည်းထားပြီး၊ တစ်ကလစ်တည်းနှင့် ပို့လွှတ်နိုင်သော စနစ်ဖြစ်ပါသည်။ 

**အသုံးဝင်မှု:** Brand များစွာကို Social Media Platform 10+ များတွင် စီမံခန့်ခွဲရာတွင် အလွန်အသုံးဝင်ပါသည်။

## ဘာကြောင့် လိုအပ်သလဲ?

**ပြဿနာ:**  
သင့်တွင် "PT", "Company A", "Personal" စသည့် Brand များစွာရှိပြီး၊ Facebook, TikTok, YouTube, Instagram အစရှိသည့် Platform တစ်ခုချင်းစီတွင် Account 10+ စီရှိနေသည်။ Post တင်တိုင်း Channel များကို လက်ဖြင့် ရွေးချယ်ရခြင်းသည် အချိန်ကုန်စေပါသည်။

**ဖြေရှင်းချက်:**  
"PT Profile" တစ်ခုဖန်တီးထားပြီး PT အမှတ်တံဆိပ်ပါသော Channel အားလုံးကို ထည့်သွင်းထားပါ။ ထို့နောက် "PT Profile" ကို တစ်ကလစ်နှိပ်ရုံဖြင့် PT Channel အားလုံးသို့ တစ်ခါတည်း ပို့လွှတ်နိုင်ပါသည်။

## အဓိက Features များ

### ✨ Quick Select (မြန်ဆန်စွာ ရွေးချယ်ခြင်း)
- Profile button ကို တစ်ကလစ်နှိပ်ရုံဖြင့် Channel အားလုံး auto-select ဖြစ်သည်
- အရောင်၊ Icon များဖြင့် မြင်သာစွာ ခွဲခြားနိုင်သည်
- Channel အရေအတွက်ကို တစ်ချက်မြင်နိုင်သည်
- Default profile ကို စာမျက်နှာဖွင့်သည်နှင့် auto-load လုပ်သည်

### 🎨 Customization (စိတ်ကြိုက်ပြင်ဆင်ခြင်း)
- **Name:** Profile အမည် (ဥပမာ: "PT Brand", "ကိုယ်ပိုင်", "ကုမ္ပဏီတရားဝင်")
- **Description:** မှတ်ချက် (Optional)
- **Color:** အရောင် 8 မျိုးမှ ရွေးချယ်နိုင်သည်
- **Icon:** Icon 6 မျိုးမှ ရွေးချယ်နိုင်သည်
- **Default:** Default အဖြစ် သတ်မှတ်နိုင်သည် (auto-select အတွက်)

### 🔒 လုံခြုံရေး Features
- Database က Channel ID များကို စစ်ဆေးသည်
- မိမိပိုင်ဆိုင်သော Channel များသာ ထည့်သွင်းနိုင်သည်
- Channel ဖျက်လိုက်လျှင် profile မှလည်း auto ဖယ်ရှားသည်
- ပိုင်ဆိုင်မှုကို အမြဲစစ်ဆေးသည်

## အသုံးပြုနည်း

### 1. Profile အသစ် ဖန်တီးခြင်း

**အဆင့် 1:** Create Post စာမျက်နှာသို့ သွားပါ

**အဆင့် 2:** Platform များရွေးချယ်ပါ (ဥပမာ: Facebook, TikTok, YouTube)

**အဆင့် 3:** "**+ New**" သို့မဟုတ် "**Create Profile**" ကို နှိပ်ပါ

**အဆင့် 4:** အချက်အလက်များဖြည့်ပါ:
- **Profile Name:** ဥပမာ: "PT Brand"
- **Description:** ဥပမာ: "Platform အားလုံးရှိ PT Channel များ"
- **Color:** အရောင် ရွေးချယ်ပါ
- **Icon:** Icon ရွေးချယ်ပါ

**အဆင့် 5:** **Channel များရွေးချယ်ပါ** - ဒီ Profile မှာ ပါစေလိုသော Channel များကို ခြစ်ပါ

**အဆင့် 6:** **Set as Default** (Optional) - Post တင်တိုင်း auto-select ဖြစ်စေချင်လျှင် ON လုပ်ပါ

**အဆင့် 7:** "**Create Profile**" နှိပ်ပြီး သိမ်းဆည်းပါ

### 2. Profile အသုံးပြုခြင်း

**အဆင့် 1:** Create Post စာမျက်နှာသို့ သွားပါ

**အဆင့် 2:** Platform များရွေးချယ်ပါ

**အဆင့် 3:** "**Quick Select Profiles**" အပိုင်းတွင် သင့် Profile များကို မြင်ရမည်:
- **Manual** - လက်ဖြင့် ရွေးချယ်ရန်
- **သင်၏ Profiles များ** - သိမ်းဆည်းထားသော Profile များ

**အဆင့် 4:** Profile button ကို နှိပ်ပါ - Channel များ auto-select ဖြစ်သွားမည်

**အဆင့် 5:** Post ကို ပုံမှန်အတိုင်း ဆက်တင်ပါ

### 3. Mode ပြောင်းခြင်း

**Profile Mode:**
- Profile button တစ်ခုခုကို နှိပ်လိုက်ရုံပါပေ
- ထို Profile ရှိ Channel အားလုံး auto-select ဖြစ်သည်
- ရွေးချယ်ထားသော Profile မှာ checkmark ပါသည်

**Manual Mode:**
- "**Manual**" button ကို နှိပ်ပါ
- သို့မဟုတ် Channel များကို ကိုယ်တိုင် check/uncheck လုပ်ပါ
- Channel ရွေးချယ်မှုကို အပြည့်အဝ ထိန်းချုပ်နိုင်သည်

## နမူနာ အသုံးပြုမှုများ

### နမူနာ 1: Brand များစွာ စီမံခန့်ခွဲခြင်း

**သင့်အခြေအနေ:**
- Brand 1: "**PT**" (Facebook 10 pages, TikTok 10 accounts, YouTube 10 channels)
- Brand 2: "**Company A**" (Facebook 5 pages, Instagram 3 accounts)
- Brand 3: "**ကိုယ်ပိုင်**" (Platform တစ်ခုလျှင် 1 account)

**ဖြေရှင်းနည်း:**
- "**PT Profile**" - PT Channel 30+ ထည့်ရန်
- "**Company A Profile**" - Company A Channel 8 ခုထည့်ရန်
- "**ကိုယ်ပိုင် Profile**" - ကိုယ်ပိုင် Channel 5 ခုထည့်ရန်

**ပို့လွှတ်ခြင်း:**
- "PT Profile" နှိပ်ရုံ → PT Channel 30+ သို့ တစ်ခါတည်း ပို့သည်
- "Company A Profile" နှိပ်ရုံ → Company A Channel များသို့ ပို့သည်
- "ကိုယ်ပိုင် Profile" နှိပ်ရုံ → ကိုယ်ပိုင် Channel များသို့ ပို့သည်

### နမူနာ 2: Campaign အခြေခံ

**သင့်အခြေအနေ:**
- ကုန်ပစ္စည်းသစ် မိတ်ဆက်ရန် Campaign လုပ်နေသည်
- Engagement မြင့်သော Channel များသာ ရွေးချယ်ချင်သည်

**ဖြေရှင်းနည်း:**
- "**High Engagement Profile**" - စွမ်းဆောင်ရည်ကောင်းသော Channel များ
- "**Full Reach Profile**" - Channel အားလုံး
- "**Test Profile**" - စမ်းသပ်ရန် Channel 1-2 ခု

**ပို့လွှတ်ခြင်း:**
- "Test Profile" ဖြင့် အရင်စမ်းကြည့်သည်
- "High Engagement Profile" ဖြင့် လွှင့်သည်
- နောက်ပိုင်း "Full Reach Profile" ဖြင့် ထပ်လွှင့်သည်

### နမူနာ 3: ဘာသာစကား အခြေခံ

**သင့်အခြေအနေ:**
- အင်္ဂလိပ် Content → အပြည်ပြည်ဆိုင်ရာ Channel များ
- မြန်မာ Content → ပြည်တွင်း Channel များ

**ဖြေရှင်းနည်း:**
- "**International Profile**" - အင်္ဂလိပ်ဘာသာ Channel များ
- "**Myanmar Profile**" - မြန်မာဘာသာ Channel များ

**ပို့လွှတ်ခြင်း:**
- အင်္ဂလိပ် Post → "International Profile" နှိပ်သည်
- မြန်မာ Post → "Myanmar Profile" နှိပ်သည်

## အကျိုးကျေးဇူးများ

### ⚡ အချိန်ကုန်သက်သာစေသည်
- Profile တစ်ခုကို တစ်ကြိမ်ဖန်တီးထား → အကြိမ်ပေါင်းများစွာ အသုံးပြုနိုင်သည်
- Channel 30+ ကို လက်ဖြင့်ရွေးရမည့်အစား → Button တစ်ခုနှိပ်ရုံ

### 🎯 အမှားလျှော့စေသည်
- ကြိုတင်သတ်မှတ်ထားသော Channel များသာ select ဖြစ်သည်
- လိုချင်သော Channel ကျန်ခဲ့မည့် အန္တရာယ် မရှိ

### 📊 ပိုမိုစနစ်တကျ ဖြစ်စေသည်
- Brand/Campaign အလိုက် စုစည်းထားနိုင်သည်
- အရောင်၊ Icon များဖြင့် လွယ်ကူစွာ ခွဲခြားနိုင်သည်

### 🚀 Scalable (ချဲ့ထွင်နိုင်သည်)
- Profile အကန့်အသတ်မရှိ
- တစ် Profile မှာ Channel အရေအတွက် အကန့်အသတ်မရှိ

## အသုံးဝင်သော Tips များ

### ✅ လုပ်သင့်သည်များ

1. **ရှင်းလင်းသော အမည်များပေးပါ**
   - ✅ "PT Brand - Channel အားလုံး"
   - ❌ "Profile 1"

2. **အရောင်ကို စနစ်တကျ သုံးပါ**
   - အပြာ → ကုမ္ပဏီ
   - အစိမ်း → ကိုယ်ပိုင်
   - အနီ → စမ်းသပ်ရန်

3. **Default Profile သတ်မှတ်ပါ**
   - အများဆုံးသုံးသော Profile ကို default လုပ်ထားပါ
   - စာမျက်နှာဖွင့်သည်နှင့် auto-select ဖြစ်သည်

4. **Test Profile ဖန်တီးထားပါ**
   - အန္တရာယ်နည်းသော Channel 1-2 ခုထည့်ပါ
   - Post များကို အရင်စမ်းသပ်ရန် အသုံးပြုပါ

### ❌ မလုပ်သင့်သည်များ

1. **Profile များစွာကို default မလုပ်ပါ**
   - Default ဖြစ်သော Profile တစ်ခုတည်းရှိနိုင်သည်

2. **ဆက်သွယ်မထားသော Channel များ မထည့်ပါ**
   - Database က အလိုအလျောက် ပယ်ဖျက်သည်

3. **Profile အမျှားမဖန်တီးပါ**
   - လိုအပ်သလောက်သာ ဖန်တီးပါ
   - မလိုတော့သော Profile များကို ဖျက်ပါ

## အဖြေရှာခြင်း (Troubleshooting)

### Profile မသိမ်းဆည်းနိုင်ပါ
**အကြောင်းရင်း:** ရွေးချယ်ထားသော Channel များ ရှိနေဦးမှာလား?  
**ဖြေရှင်းချက်:** ဆက်သွယ်မထားတော့သော Channel များကို ဖယ်ရှားပြီး ထပ်ကြိုးစားပါ

### Channel များ auto-select မဖြစ်ပါ
**အကြောင်းရင်း:** Profile ကို ရွေးချယ်ထားသလား? (highlight ဖြစ်နေသလား)  
**ဖြေရှင်းချက်:** Profile button ကို ထပ်နှိပ်ကြည့်ပါ

### Default Profile auto-load မဖြစ်ပါ
**အကြောင်းရင်း:** Browser cache ရှင်းထားပြီးပြီလား?  
**ဖြေရှင်းချက်:** စာမျက်နှာကို refresh လုပ်ပါ (Ctrl+R သို့မဟုတ် F5)

### Profile ကို မဖျက်နိုင်ပါ
**အကြောင်းရင်း:** သင်ဖန်တီးထားသော Profile လား?  
**ဖြေရှင်းချက်:** ဖန်တီးသူသာ ဖျက်နိုင်သည်

## အကူအညီ လိုအပ်ပါက

ပြဿနာတစ်ခုခု ကြုံတွေ့ပါက:

1. Browser Console တွင် error message များကို စစ်ကြည့်ပါ
2. Backend API running ဖြစ်မဖြစ် စစ်ပါ (`pm2 list`)
3. Database connection ကို စစ်ပါ
4. Browser DevTools ရှိ Network tab ကို စစ်ပါ
5. အသေးစိတ် error message များနှင့်အတူ ဆက်သွယ်ပါ
